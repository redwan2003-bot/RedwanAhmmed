# Technology Icons Mapping

To match the reference site, please add `.svg` files for each technology in this folder. The `TechMarquee` component is configured to look for these filenames:

| Technology | Filename |
| :--- | :--- |
| React | `react.svg` |
| Next.js | `nextjs.svg` |
| TypeScript | `typescript.svg` |
| Node.js | `nodejs.svg` |
| Angular | `angular.svg` |
| Tailwind CSS | `tailwind.svg` |
| Framer Motion | `framer.svg` |
| Three.js | `threejs.svg` |
| Express | `express.svg` |
| MongoDB | `mongodb.svg` |
| PostgreSQL | `postgresql.svg` |
| GraphQL | `graphql.svg` |
| Docker | `docker.svg` |
| AWS | `aws.svg` |
| Vercel | `vercel.svg` |
| Figma | `figma.svg` |
| Redux | `redux.svg` |
| Jest | `jest.svg` |

The component will automatically render these icons next to the text once you place them here.
